# Lab Insight AI 🧪🤖

## Live Application

https://lab-insight-ai-hdg3.vercel.app/

## Overview
Lab Insight AI is an AI-powered clinical laboratory assistant designed to help biochemistry and medical students understand laboratory tests, concepts, and result interpretations. The app provides simple AI explanations and learning support to make clinical laboratory education easier and more accessible.

## Problem It Solves
Many students struggle to understand complex laboratory concepts, test principles, and clinical interpretations. Lab Insight AI solves this problem by providing an interactive AI assistant that explains laboratory topics in a student-friendly way.

## Features
- AI-powered clinical laboratory assistant
- Explanation of laboratory tests and concepts
- Simple interpretation support for learning purposes
- Student-friendly dashboard
- Interactive learning support
- Clean and responsive user interface

## AI Feature
The app includes an AI assistant that helps students learn clinical laboratory concepts.

### AI System Instruction
"You are a clinical biochemistry tutor. Explain laboratory tests and concepts in simple language for students. Provide clear, accurate, and educational explanations."
